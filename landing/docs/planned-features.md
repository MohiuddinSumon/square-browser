# SquareBrowser — Planned Features

This document tracks feature ideas and planned work beyond the shipped foundation. Items move to GitHub Issues when actively scheduled.

---

## Shipped (v1.x)

- Full-featured Android browser (tabs, bookmarks, history)
- Daily browsing time limits (30m / 1h / 2h / 3h / 6h presets)
- Strict Mode — hard lock until midnight, no override
- Permanent, undeletable local browsing history
- Per-domain usage statistics
- Built-in ad blocker (CSS selector injection)
- Dark / light theme with persistence
- Long-press link context menu
- Cloudflare verification handling

---

## In Progress

### Analytics & Insights
- Weekly and monthly usage trend views
- Streak tracking (days under limit)
- "Most visited" domain summary

### Controls
- Custom domain block lists (user-defined)
- Scheduled browsing windows (e.g. no browsing before 9am)
- Focus mode with site whitelists (only allow specific domains)

### Discoverability
- Home screen timer widget (always-visible countdown)
- Quick settings tile for enabling Strict Mode

---

## Planned

### iOS
- iOS release (SwiftUI + WKWebView)
- Feature parity with Android version

### Family & Parental Controls
- Supervised profiles with PIN-locked settings
- Parent dashboard to review child's usage stats
- Separate per-profile limits

### Accountability Partner
- Share weekly usage summary with a trusted contact (opt-in, local export)
- Custom lock screen messages from your future self

### Reporting
- Exportable usage reports (CSV / PDF)
- Weekly email digest (optional, local generation only — no server)

---

## Future / Exploratory

### Community
- Anonymous usage benchmarks ("you browse 40% less than average")
- Opt-in community accountability groups

### Enterprise & Education
- Managed deployment for workplaces and schools
- Admin policy enforcement (minimum limits, mandatory Strict Mode)
- Bulk licence / MDM compatibility

### Platform
- Open REST API for third-party integrations (habit trackers, productivity apps)
- Research partnership data sharing (fully anonymised, opt-in only)
- Browser extension for desktop (Chrome / Firefox)

3. **Family / Parental plan** — subscription for multi-profile managed mode
4. **Enterprise tier** — workplace/school deployment with admin controls
